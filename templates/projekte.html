BLOG

Basierend auf Django 4.1.6 ein Blog, in dem Leser:innen pseudonymisiert kommentieren können. Die Beitragslinks werden mit der Slug-Funktion, basierend auf dem Beitragstitel, generiert.

Für jeden Beitrag kann optional ein Bild hochgeladen werden. Beim UI habe ich mich bei Bootstrap bedient.

Projektlink GitHub

ANMESY

Ein einfaches, anonymes Hinweisgebersystem in Python und Django. Der Name setzt sich zusammen aus ANonymes MEldeSYstem und hat, aufgrund der Konfiguration, eine gewollte Nähe zu den Worten Amnesie und Amnestie. Die Meldungen selbst sind über den Adminbereich einsehbar.

Benutzt werden Crispy Forms, aber selbstverständlich ist das ganze auch mit den native forms von Django nutzbar.

Projektlink GitHub

DOKUSYS

Ein einfaches Dokumentationssystem für Sozialarbeiter:innen für die nachvollziehbare Arbeit mit ihren Klient:innen. In das Formular für den jeweiligen Eintrag ist auch ein Bildupload integriert. Die Demoversion funktioniert ohne Anmeldung, während die WebApp in der Production nur authentifizierte Nutzer:innen Lese- und Schreibberechtigung gibt, die vorher von einer Nutzer:in mit entsprechenden Administrationsrechten freigegeben wurden.

Auch dieses Projekt lässt sich zu 100% an die unterschiedlichsten Bedürfnisse anpassen. Eine Überlegung könnten automatisierte Auswertungen und Übergabe in die Statistik während oder nach dem Betreuungsende sein.
